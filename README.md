
# 🛒 ShopX Admin Panel

A modern and responsive **E-commerce Admin Dashboard UI** built using **HTML, CSS, and Bootstrap 5**.  
This project provides a clean and professional interface for managing an online store.

---

## 🚀 Features

- 📊 Dashboard overview (Revenue, Orders, Customers, Stock)
- 📦 Product & Order Management UI
- 👥 Customer section layout
- 📈 Sales Analytics (Monthly chart & category split)
- ⚡ Quick Action buttons
- 🔔 Notification & Search UI
- 🌙 Dark Theme Design
- 📱 Fully Responsive Layout
- 🎨 Modern UI with custom styling

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (DM Sans & Space Mono)
